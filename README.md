Develop by [Eri](https://github.com/ryonakae/eri).


## Setup
```
$ brew cask install vagrant vagrant-manager

$ vagrant plugin install vagrant-hostsupdater

$ vagrant box add vccw-team/xenial64

$ vagrant up
```

More information: http://vccw.cc/


## Develop
```
$ npm install

$ npm run start
```
